## نحوه شروع (برای توسعه‌دهنده جدید)

1. **مطالعهٔ نقشه راه:** ابتدا فایل `Pathway.md` را بخوانید تا درک کلی از مسیر پروژه پیدا کنید.
2. **بررسی فاز جاری:** اگر قصد همکاری در فاز خاصی را دارید، فایل مربوط به آن فاز را در پوشه `/phases` مطالعه کنید. این فایل‌ها شامل جزئیات گام‌به‌گام، ورودی‌ها، خروجی‌ها و معیار پایان آن فاز هستند.
3. **مطالعه مستندات:** پس از شروع هر فاز، مستندات خروجی آن در پوشه `/docs` تولید و به‌روزرسانی می‌شوند. این مستندات مرجع اصلی برای فازهای بعدی هستند.
